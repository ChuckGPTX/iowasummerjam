# Iowa Summer Jam

Next.js site for Iowa Summer Jam, a music festival experience with artist lineup, tickets, media, sponsor information, and event storytelling.

## What It Includes

- Home page with hero, lineup, sponsors, and video recap sections
- About, artist, media, sponsors, and tickets pages
- Ticket form UI
- Sponsor and media gallery components
- Custom Iowa Summer Jam visual styling

## Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Radix UI primitives
- lucide-react icons

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Portfolio Status

This is a public event-site prototype. Before pinning it, verify that event dates, ticket details, sponsor names, and media assets are current and approved for public use.
